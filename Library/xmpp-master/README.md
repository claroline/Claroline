xmpp
====

为xmpphp 添加了如下的新的方法：

registerNewUser            //注册一个新用户

addRosterContact           //发送添加好友的请求

accept friend request      //接受好友请求

deleteRosterContact        //删除某个好友

roomMessage                //发送群聊消息

createChatRoom             //创建群聊

kickUserOutToChatRoom      //把某个人剔除群聊
